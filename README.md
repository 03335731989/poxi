### Left features:
 - Layers
 - Animations
 - Ellipse insertion

### Left tools:
 - Eraser (Delete tiles)
 - Bucket (Fill enclosed tile area)
 - Dropper (Extract tile color)
 - Cropper (Crop out tile area)
