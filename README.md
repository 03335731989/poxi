### Coming soon:
 - Layers
 - Animations
 - Ellipse insertion

### API:
  - drawImage (ctx, x, y)
  - strokeRect (x, y, width, height, color)
  - fillRect (x, y, width, height, color)
  - applyColorLightness (batch, factor)
  - applyPixelSmoothing (batch)
  - getRandomRgbaColors () => Array
